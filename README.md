
# Интернет магазин

## Описание проекта
Данный проект представляет собой полностью функционирующий интернет магазин со списком товаров, детальным представлением товаров, корзиной покупок с возможностью редактирования, системой купонов, полным процессом оформления заказа, оплатой с помощью интегрированного платежного шлюза Stripe и панелью администратора.

## Установка и запуск проекта
- Клонирование репозитория
````
git clone https://github.com/kamilgareev/shop
````
- Создание виртуального окружения
````
python -m venv venv
````
- Активация виртуального окружения
````
venv\Scripts\activate
````
- Установка необходимых зависимостей 
````
pip install -r requirements.txt
````
- Установка параметров среды в файле .env
````
#Общая информация
SECRET_KEY = '...'
DEBUG_STATUS = ...
ALLOWED_HOSTS = '...'
TIME_ZONE = '...'

#База данных
DB_ENGINE = '...'
DB_NAME = '...'
DB_USER = '...'
DB_PASSWORD = '...'
DB_HOST = '...'
DB_PORT = ...

#Stripe
STRIPE_PUBLISHABLE_KEY = '...'
STRIPE_SECRET_KEY = '...'
STRIPE_API_VERSION = '...'
````
